# MySQL Solutions

Bu klasörde MySQL ile çözülen LeetCode soruları bulunmaktadır.

## Format

Her dosya şu formatta adlandırılır:
- `{soru-numarası}-{soru-başlığı}.sql`

Örnek: `0175-combine-two-tables.sql`

## İçerik

Her dosya şunları içerir:
- Soru açıklaması (yorum olarak)
- Çözüm
- Açıklama (gerekirse)
