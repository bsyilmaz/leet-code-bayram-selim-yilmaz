# LeetCode Solutions

Bu repository, LeetCode platformunda çözdüğüm soruların çözümlerini içermektedir.

## Klasör Yapısı

- `mysql/` - MySQL ile çözülen sorular
- `python/` - Python ile çözülen sorular
- `javascript/` - JavaScript ile çözülen sorular
- `java/` - Java ile çözülen sorular

## Nasıl Kullanılır

Her soru için ayrı bir dosya oluşturulur. Dosya adı soru numarası ve başlığından oluşur.

Örnek: `mysql/0175-combine-two-tables.sql`
